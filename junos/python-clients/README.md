Common Uses
-----------

### Configuration

-	Create a firewall policy
-	Add an IP on an interface
-	Add a VLAN to a device
-	Apply a VLAN to an interface
-	Enable a routing protocol on an interface

### Operations

-	Fetch a request support information (RSI) from a device

### Maintenance

-	Upgrade a device
-	Clean up log files
-	Restart a service
-	Reboot a device
-	Clear counters
